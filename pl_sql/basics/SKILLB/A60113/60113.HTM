Oracle PL/SQL: Basics

Course ID: 60113

Course Overview

This course introduces participants to the basics of the PL/SQL language. Participants will learn how to build a simple PL/SQL block as well as how to communicate with the database in order to retrieve and manipulate data. This course also covers conditional control processing and looping constructs in a PL/SQL block, along with the features and application of cursors. Additionally, participants will learn the basics of the Oracle Procedure Builder tool.

Learn To:

  • Identify the basic concepts of PL/SQL and the PL/SQL program constructs.
  • Handle PL/SQL variables and datatypes.
  • Identify the programming guidelines for Procedure Builder components.
  • Identify the uses of Procedure Builder components.
  • Access the database using PL/SQL.
  • Conditionally control processing in a PL/SQL block.
  • Iterate statements using various types of loops.
  • Identify the attributes and applications of explicit cursors.

Content Emphasis

Skills-Based

Audience

Application Developers, Database Administrators, and Tech Support Professionals. Participants should have knowledge equivalent to: Course 60111, Oracle SQL and SQL *Plus: Basic SELECT Statements; and, course 60112, Oracle SQL and SQL *Plus: DML and DDL.

Total Learning Time

7 - 8 Hour(s)

Course Contents

Unit 1: PL/SQL: An Introduction

Duration: 2 Hour(s)
  • Identify the definition and features of PL/SQL.
  • Identify the benefits of PL/SQL.
  • Identify the process involved in processing a PL/SQL block of code.
  • Identify the basic layout and structure of a PL/SQL block.
  • Identify and compare the PL/SQL program constructs that are based on the PL/SQL block structure.
  • Identify the syntax rules to apply when creating a PL/SQL block and PL/SQL identifiers.

Unit 2: Developing a Simple PL/SQL Block

Duration: 2 Hour(s)
  • Identify the guidelines for handling PL/SQL variables.
  • Identify the syntax and guidelines to declare PL/SQL variables and constants.
  • Identify the datatypes supported in PL/SQL.
  • Identify the features of scalar datatypes.
  • Identify the categories and subtypes of scalar datatypes.
  • Declare scalar variables using the %TYPE attribute.
  • Identify the features and structure of PL/SQL Tables.
  • Identify the syntax to create PL/SQL Tables.
  • Identify the features and structure of PL/SQL Records.
  • Identify the syntax to declare PL/SQL Records.
  • Declare PL/SQL Records using the %ROWTYPE attribute.
  • Identify the syntax of a PL/SQL assignment statement.
  • Identify the valid operators available in PL/SQL expressions.
  • Identify the valid SQL functions available in PL/SQL expressions.
  • Identify the valid SQL conversion functions available in PL/SQL expressions.
  • Identify the rules to reference non-PL/SQL variables in PL/SQL expressions.
  • Identify the features of nested PL/SQL blocks and the rules of variable scoping in nested PL/SQL blocks.
  • Identify the guidelines for writing consistent and clear PL/SQL code.
  • Identify the guidelines for creating unique identifier names.
  • Identify the features and components of Procedure Builder.
  • Identify the uses and components of the Program Unit Editor in Procedure Builder.
  • Identify the uses and components of the Interpreter in Procedure Builder.

Unit 3: Accessing the Database from PL/SQL

Duration: 1 - 2 Hour(s)
  • Identify the syntax and guidelines of the SELECT ... INTO statement to retrieve data from the database into scalar variables and PL/SQL records.
  • Identify the definition of exceptions and when are raised.
  • Identify the guidelines for naming PL/SQL variables in order to avoid runtime errors.
  • Insert new rows in the database from a PL/SQL block, using the INSERT command.
  • Update rows in a database from a PL/SQL block, using the UPDATE command.
  • Delete existing rows in a database table from a PL/SQL block, using the DELETE command.
  • Identify the definition and types of cursors.
  • Identify the definition and uses of SQL cursor attributes.
  • Identify the effects of the COMMIT command.
  • Identify the effects of the ROLLBACK command.
  • Save a partial transaction with the SAVEPOINT command.

Unit 4: Controlling Flow in PL/SQL Blocks

Duration: 1 Hour(s)
  • Identify the uses and syntax of the If statement in PL/SQL.
  • Identify logic tables and build simple boolean conditions with a comparison operator.
  • Identify the syntax and structure of a basic loop.
  • Identify the syntax and structure of a FOR loop.
  • Identify the guidelines for working with the FOR loop.
  • Identify the syntax and structure of a WHILE loop.
  • Identify the definition of nested loops and labels.
  • Using labels, exit an outer loop from an inner loop.

Unit 5: Explicit Cursors

Duration: 1 Hour(s)
  • Review the definition and types of cursors.
  • Identify the functions of a cursor.
  • Identify the guidelines and syntax for declaring an explicit cursor.
  • Identify the guidelines and syntax for opening an explicit cursor.
  • Identify the guidelines and syntax for fetching data from an explicit cursor.
  • Identify the guidelines and syntax for closing an explicit cursor that is open.
  • Identify the syntax and uses of the %ISOPEN cursor attribute.
  • Identify the syntax and uses of the %FOUND cursor attribute.
  • Identify the syntax and uses of the %NOTFOUND cursor attribute.
  • Identify the syntax and uses of the %ROWCOUNT cursor attribute.
  • Identify the syntax and benefits of creating a cursor with parameters.
  • Identify the syntax and guidelines for using the Cursor FOR loop.
  • Identify the syntax and features of the WHERE CURRENT OF clause.
